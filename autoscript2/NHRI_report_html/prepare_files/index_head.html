MuSiCa Mutational Signatures in Cancer (Github)

(DÃ­az-Gay et al., BMC Bioinformatics (2018))

It is a shiny-based web application aimed to visualize the somatic mutational profile of a series of provided samples (different formats are allowed) and to extract the contribution of the reported mutational signatures (Alexandrov L.B. et al., Nature (2013), Catalogue Of Somatic Mutations In Cancer, COSMIC (2020)) on their variation profile. It is mainly based on the MutationalPatterns R package (Blokzijl et al., Genome Medicine (2018)).



Mutation counts plot

Represents the prevalence of somatic mutations in the sample, the unit is (number of mutations per megabase).

Signatures plot/

COSMIC signatures contributions

Compare with the known mutation signatures in the Catalog of Somatic Mutations in Cancer (the table also included), and list the similar mutation signatures in the sample.

Profile 96 plot (Base substitution mutation spectra)

A row represents a subset, there are a total of 96 possible mutation types (6 substitutions * 4 5'bases * 4 3'bases), through the reference genome (ex. hg38) trinucleotide Frequency, obtain the relative proportion of mutations in this subset of samples, and draw the individual numbers of 96 mutation types.

download table here: profile96.txt

Comparison with cancer signatures

Comparison of the sample with the known characteristics and mutation characteristics reported in different human cancer types.

Reconstructed profile table

The first row displays the actual tumor profile. The second row displays the profile reconstructed from the predicted mutational signatures component in the first row. The last row displays the prediction error, which is the difference between the actual profile and the reconstructed profile.
